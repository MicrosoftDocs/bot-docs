Task<bool> Microsoft.Bot.Builder.FormFlow.Advanced.IFieldPrompt< T >.DefineAsync ( T  state)

Build the prompt and recognizer for dynamically defined fields.

Returns
True if field is defined.

This method is called before asking for Prompt. This provides an opportunity to dynamically define the field based on the current state or external information. The IFieldState<T>.Dependencies method identifies fields that this one depends on. All of them will be complete before the field will be shown to the user, but this method might be called earlier to define the field for things like status and initial matching or validation.
